Frequently Asked Questions

General Questions

What is Consyn?

Consyn is an intelligent technology platform that helps modern enterprises build, scale, and transform their businesses with AI-powered analytics, enterprise security, and seamless integrations.

How much does Consyn cost?

We offer three pricing tiers:

  • Starter: $99/month - Perfect for small teams
  • Professional: $299/month - For growing businesses
  • Enterprise: Custom pricing - For large organizations

All plans include a 14-day free trial with no credit card required.

Is there a free trial?

Yes! All plans come with a 14-day free trial. No credit card required to start.

Can I change my plan later?

Absolutely! You can upgrade or downgrade your plan at any time from your dashboard. Changes take effect immediately.

Technical Questions

What programming languages do you support?

Our API supports any language that can make HTTP requests. We provide official SDKs for:

  • JavaScript/TypeScript
  • Python
  • Go
  • Ruby

Do you support on-premise deployment?

Yes, on-premise deployment is available for Enterprise plan customers. Contact our sales team for details.

What are the API rate limits?

  • Starter: 100 requests/minute
  • Professional: 1,000 requests/minute
  • Enterprise: Custom limits

How do I get my API key?

  1. Log in to your dashboard
  2. Go to Settings ‚Üí API Keys
  3. Click "Generate New Key"
  4. Copy and securely store your key

Security & Compliance

Is my data secure?

Yes. We use:

  • AES-256 encryption at rest
  • TLS 1.3 encryption in transit
  • Regular security audits
  • SOC 2 Type II compliance

What compliance certifications do you have?

We are compliant with:

  • SOC 2 Type II
  • GDPR
  • HIPAA (Enterprise)
  • ISO 27001
  • PCI DSS

Where is my data stored?

Data is stored in secure data centers with geographic redundancy. Enterprise customers can choose their data region.

Do you perform security audits?

Yes, we conduct:

  • Annual penetration testing
  • Quarterly security audits
  • Continuous vulnerability scanning
  • Regular compliance reviews

Integrations

How many integrations do you support?

We support 1000+ integrations including popular tools like Slack, GitHub, AWS, and Google Cloud.

Can I build custom integrations?

Yes! Use our REST API or webhooks to build custom integrations. See our Integration Guide for details.

What if my tool isn't supported?

Request an integration by emailing support@consyn.tech. We prioritize based on user demand.
