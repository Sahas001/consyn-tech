Getting Started with Consyn

This guide will help you set up and start using Consyn in minutes.

Prerequisites

Before you begin, make sure you have:

  • A Consyn account (sign up for free at consyn.tech)
  • Basic understanding of REST APIs (for API integration)
  • Your API credentials (available in your dashboard)

Quick Start

Step 1: Create Your Account

  1. Visit consyn.tech
  2. Click "Start Free Trial"
  3. Fill in your details
  4. Verify your email address
üí° Pro Tip: No credit card required for the 14-day free trial!

Step 2: Connect Your Data Sources

  1. Navigate to Settings ‚Üí Integrations
  2. Select your data source from our list of 1000+ integrations
  3. Follow the authentication flow
  4. Configure your sync preferences

Step 3: Configure Your Workspace

  1. Go to Workspace Settings
  2. Set up your team members and permissions
  3. Configure your organization preferences
  4. Customize your dashboard

Step 4: Deploy Your First Workflow

  1. Click Create Workflow in your dashboard
  2. Use our visual builder to design your workflow
  3. Test your workflow in sandbox mode
  4. Deploy to production

Basic Concepts

Workspaces

Workspaces are isolated environments where you manage your projects, data, and team members. Each workspace has its own settings, integrations, and workflows.

Workflows

Workflows automate your business processes by connecting data sources, applying transformations, and triggering actions. Build complex automation with our visual drag-and-drop builder.

API Keys

API keys authenticate your requests. Keep them secure and never share them publicly. You can generate and manage API keys from your dashboard settings.

‚ö†Ô∏è Security Note: Never commit API keys to version control. Use environment variables instead.

Analytics

Real-time analytics provide insights into your data and workflow performance. Monitor key metrics, identify bottlenecks, and optimize your processes.

Example: Your First API Call

Here's a simple example to get you started with the Consyn API:

curl -X GET https://api.consyn.tech/v1/workspaces \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
